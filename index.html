WanderFlow

Turn every SteamVR game into a personal gym

A customizable SteamVR plugin that maps your in-place head/hand/foot movements to virtual locomotion. Intuitively and seamlessly.

📦

Installation

Quick setup guide

1
Use WanderFlow Installer to install the plugin
2
Restart SteamVR if it was running during installation

Driver action binding

⚠️ Required step

This step is required for WanderFlow to take effect in most SteamVR apps.

Navigate to: SteamVR settings → Controller → Manage controller bindings → Choose another

Select your current SteamVR app and WanderFlow virtual controller.

These bindings describe how WanderFlow output is used in your SteamVR app.

  • ⚠️ Extra Settings (IMPORTANT): You may need to navigate to Extra Settings tab and check "Return bindings with left hand" (or even "Return bindings with right hand") for certain apps (e. g. VRChat, Skyrim VR) to work with WanderFlow.
  • 🕹️ Joystick and trackpad position: WanderFlow uses these as movement output. Bind either joystick or trackpad position to the movement action in your SteamVR app.
  • 🏃 /input/run and joystick&trackpad click: WanderFlow uses these to signal running. Bind any one of these to the run/sprint action in your SteamVR app.
  • ⬆️ A button, X button, and /input/jump: Bind any one of these to the jump action in your SteamVR app.
  • 🪶 Grip touch and /input/glide: WanderFlow uses these to signal gliding (e.g. when emulating wing flap or swimming). This could be bound creatively to some action in your SteamVR app.
  • 🔄 /input/turnstick: You can bind this to the look/rotation action if you want to use the head tilt turn feature (see section Locomotion modifiers).
🎛️

Client action binding

Navigate to: SteamVR settings → Controller → Manage controller bindings → Choose another

Select WanderFlow Client as the app and select your current (hand) controllers.

These bindings describe how your physical controllers interact with WanderFlow.

  • Calibration action: defaulting to trigger, it can be combined with T-posing to initiate calibration (see next section Calibration).
  • Pause action: when the pause action is in effect, you cannot start locomotion using WanderFlow. On Oculus Touch and Index Knuckle, it defaults to A/B/X/Y button touch - that is, touching any of these buttons prevent locomotion from starting.
  • Start action: if this action is not bound, you can start locomotion as long as you are not pausing it. If the start action is bound, you will need to use this action to start locomotion. For example, if you bind it to thumbrest touch, you will need touch the thumbrest while moving your hand/feet to start locomotion.
  • Stop action: This action instantly stops any hand-based locomotion. On Oculus Touch and Index Knuckle, it defaults to grip press so that pressing the grip stops any on-going hand-based locomotion.
  • Stop-feet action: This action instantly stops any foot-based locomotion.

Calibration

How to calibrate

T-pose when looking straight forward and press the calibration button (default: trigger) on both controllers. If the calibration action is not bound, you can caliberate by T-posing for 0.5 seconds without pressing any button.

Calibration is required before you can use any foot-based locomotion feature. It also improves the detection and accuracy of hand-based locomotion.

👐

Hand-Based locomotion

🏃 Arm swing

Set Hand locomotion to Simple back-and-forth swing in the control panel to use this feature. Lean forward slightly, put one hand in front and one hand behind, and swing your arms back and forth to start moving. Keep palms vertical and swing fast to start running. Slow down or have palms face downward to stop running. Wear wrist weights to instensify your workout! (⚠️ Consult your doctor first, start light, and stop if you experience any discomfort)

⚙️ Sagittal hand wheel

Set Hand locomotion to Move hands in sagittal wheel motion in the control panel to use this feature. Move your hands in a circular motion as if attached to wheels on your sides. You can move the wheel in either direction corresponding to forward and backward walking. Keep palms vertical and move hands fast to start running. Slow down or have palms face downward to stop running.

🕊️ Simple flap

Set Flap to Simple flap in the control panel to use this feature. Extend your arms to your left and right, then start flapping up and down fast. Wear wrist weights when playing to rip those delts! (⚠️ Consult your doctor first, start light, and stop if you experience any discomfort)

🦅 Realistic flap

Set Flap to Emulate realistic insect/bird wing angle in the control panel to use this feature. Extend arms to your left and right, start flapping up and down. The radial front of your palm needs to tilt down in a down stroke and tilt up in an up stroke to provide propulsion - mimicking real bird wing mechanics.

🏊 Swimming

Extend both arms forward with backs of hands facing each other (hence palms are facing away from each other). Move hands away from each other to propel. Keep using frog style, or switch to freestyle swimming motion to trigger "running" mode.

👣

Foot-Based locomotion

🚶 Walk-in-place

After calibration, lean forward slightly and move your feet up and down. Small hops with both feet can activate running mode. Lean back to start walking backward. Note that this feature can even be used if your foot trackers and HMD do not share tracking coordinates (e. g. if you are using an Oculus headset with Vive trackers) with only slightly reduced accuracy.

⛸️ Roller skate

Requires calibration and unified coordinate system for the HMD and foot trackers. Slide one foot on the floor to move in the opposite direction. Alternate feet like roller skating for smooth movement.

💡

Locomotion modifiers

↔️ Head tilt strafe

When enabled, tilting your head adds sideways movement to arm swing, hand wheel, and walk-in-place locomotion.

🔄 Head tilt turn

When enabled, tilting your head while strafing causes rotation. Requires /input/turnstick bound to the look/yaw/rotation action in your SteamVR app.

Troubleshooting

🚨 Common issues

If WanderFlow client cannot detect the driver:

  • ✅ Check that WanderFlow Driver is installed in: [SteamLocation]\steamapps\common\SteamVR\drivers\wanderflow\
  • ✅ Unblock and enable WanderFlow in: SteamVR Settings → Startup/Shutdown → Manage Addons

.

Does not work! (despite that SteamVR Settings -> Controllers -> TEST CONTROLLER shows output)

  • ✅ (More details in Client action binding section) In SteamVR controller bindings UI, navigate to Extra Settings tab and enable "Return bindings with left hand" and save settings.
  • ✅ You may need to enable "Return bindings with right hand" as well.

.

Arm swing not working:

  • ✅ You may need to increase the distance between your hands, especially along the sagittal(forward-backward) axis, to trigger movement
  • ✅ You can reduce "min hand distance" in the control panel to reduce the requirement above
  • ✅ Either lean forward slightly or disable Leaning detection in the control panel

.

Walk-in-place not working:

  • ✅ Make sure that WanderFlow is caliberated - see the Calibration section
  • ✅ Either lean forward slightly or disable Leaning detection in the control panel
